# Personal_Blog
Django 를 이용한 풀스택 블로그 개발 개인 프로젝트

## 💻 프로젝트 소개
Django, Docker, AWS 를 이용한 블로그 개발 개인 프로젝트입니다.

## ⏲️ 개발 기간
06.2022 - 08.2022

### ⚙️ 개발 환경
<ul>
  <li>Django, Docker, AWS</li>
  <li>python 3.9.13</li>
  <li>IDE : Pycharm</li>
  <li>Database : Sqlite, PostgreSQL</li>
</ul>
